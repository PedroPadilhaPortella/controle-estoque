# Controle de Estoque

Sistema Web de Controle de Estoque de Produtos, Fornecedores, Categorias de Produtos, com Sistema de Login de Usuários e Acesso baseado em Níveis.

### Ferramentas
 - PHP
 - Banco de Dados MySql 
 - HTML
 - Bootstrap
 - Editor de Código Visual Studio Code
 - Apache Xampp Server
